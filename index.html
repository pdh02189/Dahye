PROJECTS
  • 식물타이쿤(PlantTycoon)
    2024.05 ~ 2024.06 (3인 팀 프로젝트)
    식물타이쿤은 현대인들을 위한 스마트 반려식물 육성 서비스입니다. 1인 가구가 늘어가는 지금, 반려식물을 통해 공허함을 해소하고자 하는 아이디어에 착안하여 개발하였습니다.

    이 프로젝트를 통해 Java 11과 Spring Framework를 심도 있게 공부하고 활용할 수 있었습니다. 또한, Mybatis를 사용한 SQL Mapper와 Oracle 데이터베이스를 통해 데이터 관리를 구현하였습니다.

    프론트엔드 부분에서는 HTML5, CSS3, jQuery, JavaScript를 사용하여 사용자 인터페이스를 개발하였고, Arduino와의 연동을 통해 식물의 상태를 모니터링하고 관리할 수 있는 기능을 구현하였습니다.

    개발 과정에서 STS, Arduino IDE, Android Studio, Visual Studio Code와 같은 다양한 IDE를 사용하였고, Notion을 통해 아이디어 회의와 프로젝트 관리도 함께 진행하였습니다.

    식물타이쿤을 개발하며 다양한 기술 스택과 도구를 활용할 수 있었을 뿐 아니라, IT 기술을 통해 실제 생활에 유용한 솔루션을 제공하는 프로젝트를 완성할 수 있었습니다.
